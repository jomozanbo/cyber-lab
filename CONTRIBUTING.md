# CONTRIBUTING

Ce lab est principalement un projet personnel.

## Comment contribuer

1. Fork le repo
2. Cree une branche : git checkout -b amelioration/nom
3. Commit : git commit -m "feat: ajout de X"
4. Push : git push origin amelioration/nom
5. Ouvre une Pull Request

## Convention de commit

feat:   Nouveau contenu ou dossier
fix:    Correction README ou script
docs:   Amelioration documentation
chore:  Maintenance
